💾 Mentés és Visszaállítás
📦 Third-Party Driver Mentés
Kimenti a rendszer összes harmadik féltől származó (third-party) driverét egy mappába. Ezt később vissza tudod állítani DISM-mel vagy pnputil-lal.
📦 ÖSSZES Driver Mentés
Kimenti az összes drivert (third-party + Windows gyári inbox driverek is). Nagyobb fájlméret, de teljes biztonsági másolat.
♻️ Lementett Driverek Visszaállítása
Egy korábban kimentett driver mappát visszatelepít az élő rendszerre (pnputil) vagy egy offline meghajtóra (DISM).
📀 ISO / WIM / ESD Alap Driver Kinyerés
Egy hivatalos Windows ISO install.wim fájljából kibányássza a tiszta gyári drivereket (alap USB, PS/2, Touchpad, stb).
🛡️ Rendszer-visszaállítási Pont
Létrehoz egy Windows rendszer-visszaállítási pontot a jelenlegi állapotról, mielőtt bármit változtatnál.
🔧 BCD Boot Hiba Javítása (BootFixer)
Ha egy Windows nem bootol (0xc0000098, BCD hiba): letölti a BootFixer.cmd boot-javító toolt a C:\DriverVarazslo mappába. Futtasd le (dupla katt, admin jogot magától kér), válaszd ki a javítandó meghajtót, és a tool újraírja a boot partíciót.

👻 Szellem- és Fantomeszközök Törlése

Eltávolítja a Registry-ből és az Eszközkezelőből (Device Manager) mindazokat a hardvereket, amelyek korábban csatlakoztatva voltak, de fizikailag már nincsenek a gépben (pl. klónozott rendszereknél megmaradt régi hardverek). Tisztább lesz tőle a Windows és gyorsulhat a rendszer.

🚀 1 Kattintásos Driver Fix

Ez a funkció automatikusan elkészíti a registry mentést (Restore Point), letiltja a Windows Gyors Rendszerindítását (Fast Startup) és az automata WU driver frissítést, törli a third-party drivereket, majd frissíti őket a legújabbakra (automatikus letöltés WU-ról), és végül újraindítja a gépet.

⚠️ FIGYELEM: WINDOWS DEFENDER

A Windows Defender "vírusnak" nézheti és futás közben gondolkodás nélkül letörölheti a Varázslót a háttérben futó scriptek és a mély rendszer-módosítások miatt. Kérlek, feltétlenül kapcsold ki a "Valós idejű védelmet", mielőtt rányomsz a Fix indítására!

🧰 Szervíz Programok — egyenkénti indítás

Egyenkénti indításnál nincs automatizálás: a program a saját beállító képernyőjével nyílik meg, minden opciót te állítasz be. A ▶ Indítás portable módban, a letöltött csomagból futtat (telepítés nélkül); ha egy program telepítve van, a telepített példány indul. Első használatkor a teljes csomag letöltődik — a folyamatot a fenti sáv mutatja.

A bejelölt programok a Program Files (x86) alá kerülnek, asztali parancsikonnal — így a gép újraindítása után is kéznél lesznek.
📊 Rendszer Riport (HTML Gépkönyv)

Egy szép, modern HTML dokumentumot generál a számítógép hardveres adatairól (CPU, Memória, Videókártyák, és Háttértárak S.M.A.R.T. adatai), amit a böngésződben megnyitva akár PDF-be is lementhetsz vagy kinyomtathatsz. A fájl a C:\DriverVarazslo mappába kerül mentésre.

🚫 Net Blokkoló Script (block.bat)

Letölti a block.bat scriptet a C:\DriverVarazslo mappába. A script - ha később futtatod - a saját mappájában (és az összes almappájában) található minden .exe kimenő internet-elérését letiltja a Windows tűzfalban. A gomb csak letölti a fájlt, nem futtatja: másold be abba a mappába, amit blokkolni akarsz, és dupla kattintással indítsd - az admin jogot magától kéri (UAC), csak fogadd el.
